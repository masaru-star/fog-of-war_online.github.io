# 「戦争の霧」オンライン
これは試作品です。以降も改善されます。

## 実行方法
ターミナルに以下のように入力
> pip install -r requirements.txt

その後、さらに以下を入力
> python app.py


ポートを公開に設定すればそのまま他の人と共有してプレイできます。

## 簡単な操作方法
※これは今後、直感的に操作できるものに変更される可能性があります。

プレイヤーの名前を決めたら部屋を作成し、IDを共有してください。

ターンは3分経過で強制終了されます。<br>
プレイヤーはユニットを動かし、領土を広げます。<br>
黄色い[○]を占領すると資源が獲得できます。
